# Shader Viewer

A fast, lightweight shader viewer designed for rapid shader development and debugging. It features hot reloading, debug tools, and a streamlined workflow to help you iterate faster.

## Key Features (Planned)

### File Management
- Load shaders from a dedicated shader folder
- Open individual shader files directly
- Cache unchanged shaders or store precompiled binaries for faster reloads

### Shader Support

#### HLSL
- Full HLSL support (vertex, compute, pixel shaders)
- Live hot reloading while editing
- Uniform interaction and inspection
- Seamless hot reloads without resetting uniform values

### Debug Tools
- Visualize mouse and UV positions in the shader
- On-screen debug overlays for useful data
- Customizable debug views (planned)

### Compatibility
- Switchable compilation options
- Ability to target different GPUs

## Future Ideas
- 3D shader preview with built-in objects or user-loaded `.fbx` models
- Geometry shader support

- Advanced 3D shader workflows
